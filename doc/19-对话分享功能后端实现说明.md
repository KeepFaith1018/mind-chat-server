# 对话分享功能后端实现说明

## 1. 功能目标

实现“对话分享”后端能力：

1. 用户可为自己的会话生成只读分享链接
2. 其他人可通过链接查看会话消息（无需登录）
3. 提供静态只读页面渲染消息内容

---

## 2. 数据模型设计

在 Prisma 中新增 `ShareableLink`：

- `id`: 分享标识（不可猜测随机串）
- `conversation_id`: 关联会话
- `created_at`: 创建时间
- `expires_at`: 可选过期时间
- `is_active`: 是否有效

关系：

- `Conversation` 新增 `shareLinks` 关系

相关文件：

- `prisma/schema.prisma`
- `prisma/migrations/20260316010000_add_shareable_links/migration.sql`

---

## 3. 后端 API

## 3.1 生成分享链接（需登录）

- `POST /v1/share`

请求体：

```json
{
  "conversationId": "uuid",
  "expireHours": 24
}
```

说明：

- 校验当前用户是否拥有该会话
- 生成随机 `linkId`
- 写入 `shareable_links`
- 返回 `shareUrl`

响应示例：

```json
{
  "linkId": "V2uB2nA7R8uQ",
  "shareUrl": "http://localhost:3001/v1/share/V2uB2nA7R8uQ/page",
  "expiresAt": "2026-03-17T01:00:00.000Z"
}
```

## 3.2 获取分享内容（公开）

- `GET /v1/share/:linkId`

说明：

- 不需要登录
- 校验链接是否存在、是否启用、是否过期
- 返回会话与消息（只读数据）

响应结构：

```json
{
  "conversation": {
    "id": "uuid",
    "title": "新对话",
    "modelId": "deepseek-ai/DeepSeek-V3",
    "createdAt": "2026-03-16T00:00:00.000Z",
    "updatedAt": "2026-03-16T00:10:00.000Z",
    "messages": [
      { "role": "user", "content": "你好", "createdAt": "..." },
      { "role": "assistant", "content": "你好，有什么可以帮您？", "createdAt": "..." }
    ]
  },
  "share": {
    "id": "V2uB2nA7R8uQ",
    "expiresAt": "2026-03-17T01:00:00.000Z"
  }
}
```

## 3.3 静态分享页（公开）

- `GET /v1/share/:linkId/page`

说明：

- 返回 `text/html`
- 服务端直接渲染只读消息列表
- 不包含输入框/发送能力

## 5. 安全策略

1. 分享 ID 使用高随机值（`crypto.randomBytes(...).toString('base64url')`）
2. 仅会话所有者可创建分享链接
3. 访问分享内容时校验 `is_active` 和 `expires_at`
4. 分享页面输出进行 HTML 转义，防止脚本注入

---

## 6. 对接建议（前端）

1. 聊天页点击“分享”调用 `POST /v1/share`
2. 复制 `shareUrl` 给他人
3. 浏览器打开 `.../page` 即可查看只读消息
